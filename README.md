# 🔭 GitVerse – HTML Preview Tool

A lightweight utility for previewing HTML files hosted on GitVerse and BitBucket, designed for developers who need a quick and clean way to view their web projects directly from a repository.

## ✨ Features

- 🌐 **Instant HTML Preview** – Render HTML files from GitVerse or BitBucket URLs
- ⚡ **Fast & Simple** – One-click preview without cloning the repository
- 🔗 **URL Rewriting** – Built-in support for clean URLs and asset redirection
- 🐳 **Vercel Ready** – Includes `vercel.json` for seamless deployment
- 📦 **Lightweight** – Minimal dependencies, easy to integrate

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or later)
- npm or yarn
